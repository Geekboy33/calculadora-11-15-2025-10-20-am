Three-Signature Verification

Every VUSD token minted requires three independent cryptographic signatures, ensuring maximum security and complete auditability.

1

DCB Treasury / DAES

USD funds are injected from DAES custody account. First blockchain signature is generated with ISO 20022 message hash.

dcbSignature: keccak256(amount, beneficiary, daesTransactionId, xmlHash)
2

Treasury Minting

Lock is accepted and moved to reserve. Second signature is generated with authorization code for tracking.

mintingSignature: keccak256(lockId, amount, firstSignature, authCode)
3

VUSDMinter Certificate

Backed certificate is created and VUSD is minted. Publication code is generated for public verification.

backedSignature: keccak256(lockReserveId, amount, firstSig, secondSig, timestamp)

AML/CFT Compliance Framework

Anti-Money Laundering & Counter-Terrorist Financing

Aligned with Wolfsberg Group Principles and FATF Recommendations

MLRO Oversight

Money Laundering Reporting Officer with unrestricted authority to investigate, escalate, block, or report any transaction. Registered with UAE FIU on goAML platform (GMLMOEC33681057391).

Transaction Monitoring

DiXiO Smart Messaging Platform for real-time AML screening. Pre-SWIFT release screening against sanctions lists, PEP databases, and adverse media sources.

Automatic alerts for: transactions >$100K, structuring patterns, 300%+ volume spikes, high-risk jurisdictions, dormant account reactivation, and pass-through behavior.

Full KYC/CDD/EDD procedures. UBO identification for 25%+ ownership. Wolfsberg AML Questionnaire for correspondent banks. Enhanced due diligence for PEPs and complex structures.

Prohibited Jurisdictions

Zero tolerance for North Korea, Iran, and sanctioned entities under UN, OFAC, EU, and UK regulations. Dynamic high-risk jurisdiction list updated quarterly.

Record Retention

All KYC/CDD/EDD records retained for minimum 10 years. Complete audit traceability and documentation integrity per Wolfsberg and FATF expectations.
